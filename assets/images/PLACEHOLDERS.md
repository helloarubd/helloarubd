# Image placeholders

This folder currently only ships `favicon.svg` (a simple rose-gold placeholder
mark) so the project runs with zero missing files.

Before you go live, add:

- `og-cover.jpg` (1200×630px) — used by the Open Graph / Twitter card tags in
  `index.html`. A photo of your jewelry, or a styled flat-lay with the Aru
  logo, works well.
- Optionally replace `favicon.svg` with your real logomark.

No code changes are required beyond dropping the files in here with the
same names — `index.html` already points to them.
